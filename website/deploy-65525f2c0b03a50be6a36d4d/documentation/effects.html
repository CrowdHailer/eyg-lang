Effect types
Effect are tracked by Eyg's type system this ensures that only those effects that are provided by an environment can be used by the program.
For example the "Alert" effect is available in the browser but code running on Arduino would have different effects availble. Ones that represent the capabilities of the system such as taking to the input output pins of that micro controller.
Shallow
Experimental implementation of shallow effect handlers. Shallow effect handlers only catch one performed effect, they need to be reapplied to catch multiple effects.
This allows stateful handling to be implemented, like the example belown.
However it makes them less ergonomic to use, particularly if for handlers that don't need to be stateful.
The default handler, described above, is a deep handler.
It will capture all performed effects.
